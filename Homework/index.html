Etiquetas más utilizadas para desarrollo web

Etiquetas para la terminal

Estas estiquetas nos ayudan al utilizar la terminal. Es recomendable usar la terminal gitbash

  • pwd: Imprime el directorio donde estamos
  • ls: Imprime archivos y carpetas donde estamos situados
  • mkdir: Crea carpetas. Poner ("") para el nombre
  • cd: Entra en las carpetas
  • cd .. : Regresa al archivo anterior
  • touch: Crea archivos ("archivo.txt")
  • rm: Elimina archivos
  • ls -help: Te dice los comandos explicados
  • code . : Abrimos la carpeta donde estabamos

Etiquetas git

Estas etiquetas te ayudan al utilizar git y subir tus cambios a github

  1. git init: Inicia repositorio local
  2. git add: Comando para que el repositorio sepa la existencia de una archivo o de sus cambios
  3. git commit -m: Almacena definitivamente los cambios y se agregan notas
  4. git push: Manda commits al servidor
  5. git pull: Recibe cambios del repositorio al local
  6. git ls -a: Ve los archivos ocultos
  7. git log: usuario
  8. git add . : Agrega todos los archivos
  9. git remore -v : Se ve donde se esta guardando

Etiquetas para HTML

Estas etiquetas nos ayudan a utilizar HTML y darle formato a nuestra pagina web

  • html: Raiz de un documento HTML. Engloba todo el contenido
  • head: Contiene metadatos/información para el documento, como el titulo, enlaces y hojas de estilo
  • title: Define el titulo del documento, que se muestra en la barra de titulo del navegador o en las pestañas
  • body: Representa el contenido principal de un document HTML, visible para el usuario
  • h1, h2, h3 --- h6: Definen encabezados
  • a: Define un hipervinculo a otra pagina web o recurso
  • img: Incrusta una imagen en el documento
  • div: Define una división o sección
  • span: Agrupa elementos en linea
  • ul: Lista desordenada (con viñetas)
  • ol: Lista ordenada (Con numeros)
  • Define el item de la lista (ul, ol, menu)
  • br: Inserta salto de linea simple
  • table: Define una tabla
  • tr: Fila de la tabla
  • td: Celda de la tabla
  • th: encabezado de la tabla
  • form: DEfine el formulario HTML para la entrada del usuario
  • input: DEfine un campo de entrada dentro de un formulario
  • label:Define una etiqueta para un elemento
  • button: Botón
  • select: Menu despegable
  • option: Opción dentro de un select
  • textarea: Campo de texto multilinea
  • link: Enlaza hojas de estilo externas
  • meta:Define metadatos que no pueden ser representados por otros elementos HTML
  • script: Se utiliza para insertar o hacer referencia a un script ejecutable
